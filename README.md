# 🎞️ Studio Polkabooth

🌐 **Live Demo:** https://polkabooth.vercel.app/

A small, cozy pocket of the internet designed to freeze fleeting moments into beautiful stills. Studio Polkabooth is a lightweight, frontend-only web photobooth application that mimics premium vintage sticker photo studios. It offers users full creative control over layout geometry, retro patterns, custom frames, and playful accents—all processed natively inside the browser.

---

## ✨ Key Features

* **Creative Layout Configurations:** Toggle between a classic 2-Grid Vertical Strip or a structured 4-Grid Editorial layout.
* **Generative Polkadot Textures:** Dynamically rendered background motifs utilizing custom mathematical zig-zag patterns with seamless color combinations: *Cocoa Blue*, *Espresso Peony*, and *Strawberry Matcha*.
* **Bespoke Border Options:** Complete customization to turn borders on or off, adjust padding margins (`borderThickness`), alter spacing between stills (`photoGap`), and select contemporary color palettes (*Soft Pink*, *Deep Latte*, *Soft Cream*).
* **Dual Workspaces:** Choose between a live camera capture sequence backed by a clean 3-second countdown, or manually upload local snapshots.
* **Atmospheric Color Grading:** Instant local rendering pipelines for *Original*, *Vintage* (warm sepia), and high-contrast *Black & White* filters.
* **Interactive Accent Studio:** A drag-and-drop canvas layout allowing users to populate, reposition, and layer native emoji accents over their final portrait strip before exporting high-resolution PNG prints.
